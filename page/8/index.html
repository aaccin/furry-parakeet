What’s Jekyll

Jekyll is a simple, blog aware, static site generator.

功能五花八门的CMS（内容管理系统）有时候并不是必需的，你只需要创建一个轻量级的静态网站…

当你有足够多的页面就会使复制模板文件和跨网站标记的过程变成繁琐的事。

今天，我将介绍一种简单的解决方案——Jekyll——使你可以轻而易举地创建小型网站。
